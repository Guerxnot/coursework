Чтобы запустить программу, нужно установить библиотеки, указанные в requirements.txt.

Для этого прописать в терминале:

```pip install -r "requirements.txt"```


Файл **"obscur_model.py"** создает черно-белое изображение изначальной фотографии, показывает размер отверстия камеры обскура и конечный результат на сенсоре.

Файл **"obscur_with_real_light.py"** создает цветное изображение изначальной фотографии, показывает размер отверстия камеры обскура и конечный результат на сенсоре. Также считает энергию, попадающую на сенсор.
